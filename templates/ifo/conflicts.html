What this page is

The status sweep raises a conflict whenever two or more sessions are recorded as holding the same room at once. That is usually a class that finished without anyone closing it out, still sitting on a room the next class needs.

Release the session that should not be holding the room. That frees the room immediately. The conflict below then closes by itself on the next sweep run, because the thing that caused it is gone -- there is no dismiss button, and there is not meant to be one.
