Warning: this invalidates the current poster for room {{ room.code }}. The QR code printed on it stops scanning, and the six-digit fallback code printed under it stops working, at the same moment. Anyone who tries the old poster will be told the code is invalid.

What happens when you confirm

  1. {{ room.code }} gets a brand new QR token and a brand new six-digit code. The old pair stops resolving immediately.
  2. You land on this room's poster page, showing the new codes.
  3. Print that poster and replace the one on the door for {{ room.code }}. Until you do, nobody can check in to this room.

When you should NOT rotate

Rotation is for a code that has leaked or a poster that has been tampered with. To correct a room's name, capacity or floor, edit the room instead -- editing never touches the codes, so the poster on the door keeps working.
