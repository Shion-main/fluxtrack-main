| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-04-04 | Tampering | lead-time gate | high | mitigate | Gate uses server timezone.now() + get_policy only; client-supplied time is never accepted; boundary tests at Manila 23:59/00:00 |
| T-04-01 | Elevation | withdraw ownership | high | mitigate | withdraw_modality_shift requires actor == requester AND status == PENDING, re-checked inside the transaction; AuditLog per attempt-outcome |
| T-04-06 | Tampering | forged time-move slot | high | mitigate | Time-move accepted only with F2F/Blended target and faculty_has_conflict==False; preferred_room re-validated against availability (never trust client pk) |

| T-04-09 | Repudiation | decision provenance | low | mitigate | AuditLog on submit/withdraw/reject with actor + target_id; notify() single write path for the requester/Dean notices |
